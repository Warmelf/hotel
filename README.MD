# Description

Test task. Made according to the Figma layout. HTML, CSS, JS, SASS, GULP.
Hotels that can be booked by clicking on the button, styles are applied after moving the cursor away from the card. By clicking on the card, you can cancel the reservation. Adaptive layout.

# How to start
npm i - для установки node_modules и инициализации
gulp - для запуска приложения
gulp build - для сборки приложения
